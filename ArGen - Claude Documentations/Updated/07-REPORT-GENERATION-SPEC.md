## 1. REPORT PURPOSE

### Primary Function
Transform evaluation scores into actionable intelligence for leadership decision-making.

### Audience Segmentation

**Leadership Report (Primary Deliverable):**
- Audience: CEO, COO, CTO, Department Heads
- Focus: Team aggregate performance, strategic insights, ROI justification
- Length: 8-12 pages
- Format: PDF

**Individual Report (Secondary):**
- Audience: Each participant
- Focus: Personal performance, peer comparison, improvement areas
- Length: 1-2 pages
- Format: PDF

**Benchmark Report (Future Product):**
- Audience: Industry at large (public or paid access)
- Focus: Cross-company performance trends
- Length: 15-20 pages
- Format: PDF + interactive web version

```

{{team_constraint_avg}}

{{percentile_rank}}
... [etc for all dynamic fields]
```

**Step 3: Design Specifications**

**Colors:**
- Primary: #1A2A4A (Deep Blue) — headers, key metrics
- Accent 1: #2ECCB6 (Fresh Emerald) — positive indicators, checkmarks
- Accent 2: #7B61FF (Soft Purple) — charts, visual elements
- Warning: #FF9F43 (Warm Orange) — improvement areas
- Background: #F7F9FC (Soft White)
- Text: #2C3E50 (Dark Gray)

**Typography:**
- Headings: Montserrat Bold, 24-36pt
- Subheadings: Montserrat SemiBold, 18-20pt
- Body: Inter Regular, 11-12pt
- Data/Numbers: Inter SemiBold, 14-18pt

**Layout Grid:**
- Margins: 0.75" all sides
- 12-column grid for alignment
- Consistent spacing: 0.25" between elements

**When Evaluation Batch Completes:**
