VS Code Extension

Automatically captures AI-generated code patterns as you paste them. Detects Copilot, ChatGPT, and other AI completions.

IDE Auto-capture Mac · Windows · Linux
⬇ Download .vsix Instructions
  • 1. Open VS Code → Extensions (Cmd+Shift+X)
  • 2. Click "..." → "Install from VSIX..."
  • 3. Select the downloaded .vsix file
  • 4. Press Cmd+Shift+P → "ArGen: Configure API Key"
  • 5. Enter your API key from Settings → API Keys
💡 Need an API key? Go to your Dashboard → API Keys to generate one.

Chrome Extension

Captures prompts and completions from ChatGPT, Claude, Perplexity, Gemini, and Copilot automatically.

Browser Auto-capture Chrome
⬇ Download .zip Instructions
  • 1. Download the extension zip file
  • 2. Unzip to a folder on your computer
  • 3. Open Chrome → chrome://extensions
  • 4. Enable "Developer mode" (top right)
  • 5. Click "Load unpacked" → select the unzipped folder
  • 6. Click the extension icon → enter your API key
💡 New to Chrome extensions? "Developer mode" is at the top-right of chrome://extensions.

Firefox Add-on

Same powerful AI usage capture for Firefox. Supports ChatGPT, Claude, and all major AI platforms.

Browser Auto-capture Firefox
⬇ Download .zip Instructions
  • 1. Download the extension zip file
  • 2. Unzip to a folder
  • 3. Open Firefox → about:debugging
  • 4. Click "This Firefox" → "Load Temporary Add-on"
  • 5. Select the manifest.json from the unzipped folder
  • 6. Click the extension icon → enter your API key

Safari Extension

AI usage capture for Safari on macOS. Works with ChatGPT, Claude, Perplexity, Gemini, and GitHub Copilot.

Browser Auto-capture macOS
⬇ Download .zip Instructions
  • 1. Download the extension zip file above
  • 2. Unzip to a folder on your computer
  • 3. Open Safari → Settings → Extensions
  • 4. Enable "Developer mode" in the Extensions tab
  • 5. Click the "+" button → select the unzipped .app bundle
  • 6. Click the extension toolbar icon → enter your API key
💡 macOS 12+ required. Safari 15+ supports Web Extensions. For custom builds, open Terminal and run xcrun safari-web-extension-converter with the unzipped folder.
